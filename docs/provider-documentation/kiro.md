# Get started

> Command-line interface for Kiro - build, test, and deploy from anywhere

The Kiro CLI brings the power of AI-assisted development to your terminal. Build, test, and deploy applications using natural language commands and automated workflows.

Get started with the Kiro CLI in minutes.

`curl -fsSL https://cli.kiro.dev/install | bash`

Start using Kiro CLI:

```bash
cd my-project
kiro-cli
```

## Core features

  - [Interactive Chat](https://kiro.dev/docs/cli/chat.md) — Engage with Kiro through natural conversation in your terminal.
  - [Custom Agents](https://kiro.dev/docs/cli/custom-agents.md) — Create and deploy specialized agents for your workflows.
  - [MCP Integration](https://kiro.dev/docs/cli/mcp.md) — Connect external tools and data sources through Model Context Protocol.
  - [Smart Hooks](https://kiro.dev/docs/cli/hooks.md) — Automate workflows with intelligent pre/post command hooks.
  - [Agent Steering](https://kiro.dev/docs/cli/steering.md) — Steer Kiro agent with your best practices and preferences.
  - [Auto Complete](https://kiro.dev/docs/cli/autocomplete.md) — Intelligent command completion with context awareness.

## Use cases

The Kiro CLI is perfect for:

- **Interactive Development**: Chat with Kiro directly in your terminal for instant help
- **Custom Automation**: Create specialized agents for your specific workflows
- **Team Standardization**: Use team-level best practices and preferences
- **External Integrations**: Connect tools and services through MCP servers
- **Intelligent Assistance**: Get context-aware suggestions and auto-completion
- **Headless Automation**: Run prompts non-interactively in CI/CD pipelines using [API key authentication](https://kiro.dev/docs/cli/headless.md)
- **Workflow Optimization**: Automate repetitive tasks with smart hooks

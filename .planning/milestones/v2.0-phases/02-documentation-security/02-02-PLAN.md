# Plan 02-02: DOC-03 — Remove Stale LiteLLM References

## Objective

Remove actionable LiteLLM references (setup steps, port references, install commands,
config examples for the Tier-2 LiteLLM layer) from the 9 active docs in `MASTER-C/docs/`.
Preserve historical notes that explain *why* LiteLLM was removed.

## Context

LiteLLM was removed in v2.0 because it corrupted the Anthropic SSE stream. The system
is now 2-tier: Claude Code → AIClient2API (:3000) → external providers. Nine docs still
reference LiteLLM. Reference counts:
- High (11-25 refs): ANTHROPIC_GATEWAY_SPEC.md, Troubleshooting-and-Fixes.md
- Medium (3-5 refs): ULTIMATE-GOAL.md, ARCHITECTURE.md
- Low (1 ref each): TESTING.md, GETTING-STARTED.md, Model-Guide.md, CONFIGURATION.md, DEVELOPMENT.md

**Rule:** Remove steps/examples/port refs that imply LiteLLM is still running. Keep sentences
like "LiteLLM was removed in v2.0 because it corrupted SSE streams."
